# ChatBuddy

A simple, responsive chatbot demo built with HTML, CSS, and JavaScript.

Features
- Clean modern UI with soft colors and rounded bubbles 
- Typing indicator, timestamps, scroll-to-bottom behavior 
- Sticky input box, Enter to send, Shift+Enter for newline 
- Dark mode toggle, responsive layout, smooth animations 
- Friendly bot behavior with clarifying questions and context 

How to run
1. Open `index.html` in your browser.

Notes :
- The bot does NOT send an automatic initial greeting, the first message should come from the user and the bot will respond only after user input.
